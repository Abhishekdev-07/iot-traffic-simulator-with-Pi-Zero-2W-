# iot-traffic-simulator-with-Pi-Zero-2W-
A comprehensive IoT traffic control system featuring a Flask web server that synchronizes real-time CSS car animations with a physical LED circuit. This project includes a custom-designed 3D-printed enclosure (CAD), a soldered protoboard interface, and a responsive web dashboard for remote operation via Raspberry Pi.
